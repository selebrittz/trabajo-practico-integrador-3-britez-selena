# Trabajo Práctico Integrador III – Frontend en React

Este proyecto es una aplicación frontend hecha con React que se conecta a una API REST ya creada.  
Permite registrar usuarios, iniciar sesión y gestionar tareas propias (crear, listar, editar y eliminar).  
También incluye páginas protegidas, validaciones básicas y uso de cookies para la sesión.

## Tecnologías utilizadas
- React (Vite)
- react-router
- Bootstrap
- Fetch API

## Funcionalidades principales
- Registro de usuarios
- Login con cookies (credentials: include)
- Cierre de sesión
- Página Home con bienvenida
- Página Profile con datos del usuario logueado
- CRUD de tareas:
  - Listar tareas del usuario
  - Crear nuevas tareas
  - Editar tareas existentes
  - Eliminar tareas
  - Marcar como completadas
